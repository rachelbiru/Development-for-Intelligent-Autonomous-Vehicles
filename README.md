# Development for Intelligent Autonomous Vehicles
## Path Planning – A* vs Dijkstra

This repository contains a simple pathfinding project implemented in Python.

The project compares:
- A* algorithm
- Dijkstra algorithm

Both algorithms are visualized on the same grid for comparison.

---

## Features
- Grid-based path planning
- Obstacle handling
- A* with heuristic
- Dijkstra without heuristic
- Visual comparison using matplotlib

---

## Algorithms
- **A\*** – heuristic-based shortest path algorithm
- **Dijkstra** – classic shortest path algorithm

---

## Technologies
- Python 3
- matplotlib
- Git & GitHub
